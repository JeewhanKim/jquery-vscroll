#jQuery-VScroll.js


##Introduction
jquery-vscroll is simple and light parallax scrolling based on jquery-parallax by Ian Lunn.

##Example
```javascript
$(function(){
  $('#first').VScroll(0.1);
});
```

##Dependencies
jQuery 1.10
